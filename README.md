# GGJ2020 #

A game for Global Game Jam 2020.

### How to Build ###

Have [node v8 and npm v5](https://nodejs.org).

```
node --version
npm --version
```

Build to [localhost:8080](http://localhost:8080).

```
npm install
npm start
```

### How to Deploy ###

Deploy to [github.io](https://ehgoodenough.github.io).

```
npm run deploy
```
